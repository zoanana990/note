## 二元搜尋 (Binary Search)
使用時機：
1. 遇到已經排序的陣列
2. 面試官想要比 `O(n)` 還要小
3. 找到陣列的分割位置
4. 找到最大最小值使得某個條片被滿足

複雜度
- 時間複雜度 `O(log(n))`
- 空間複雜度 `O(1)`
